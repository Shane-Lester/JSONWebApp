The help page

This should help guide you through making your own data

1)Loading your data

The pages have been populated with the James Cook ENT data for you to uses as a template.

You can load up other data sources if they are in the correct format.

You can create the two files you need for the app -these accessed by the tabs at the top of the screen - Department and Clinical

If you navigate away without being saved, any changes you have made will be lost


You can now upload a file from your PC to the webApp for editting (before downloading it and saving it again)

Uploading the data

The Department tab will only let you upload a department.json file

The Clinical tab will only let you upload a clinical.json file

The file must have the appropriate formatting or the upload won't work ie it should have been created with the webApp

The first and last parts of the URL are standard- so they are automatically added

You need to fill in the root of the address (same as for the app) and the specialty (defaults to ent)

Ensure your URL's are all lowercase

Therefore- you can save your work (see- 5) below and reload it to carry on later


2)Editting an item

If you wish to edit an item -click on the blue pencil icon

If you wish to delete an item -click on the red trash icon

You can change the order of the items with the up and down arrows

You can add an whole new item by filling in the title in the 'new item title' box and pressing the green add button

- the item can them be editted, moved and deleted

Each item is a title and information

The Clinical items are more structured, the Department items are more flexible

You can't add pictures but that is coming soon


3)Clinical - clinical.json file -see tab above

You can change the item title in the edit screen

Fill in all of the boxes and click either the Cancel or Save buttons

Each box can hold as much text as you wish. It will not allow you to put newline/breaks in.

If you want a list/ newline type <br> to add a break


4)Department - department.json file -see tab above

This page can be made up of as many sections as you wish

The page will only show the boxes that you fill

The Heading and Subheading are simply bigger text

Info1 and Info2 are information boxes for text.

At the end of each section an horizontal line will automatically be added as a divider

If you want more information, simply add more sections

Click Cancel or Save to finish

Coming Soon -Bold and Underline options


5) Saving your data

Press the download button when you want to save your data

This will CREATE A DOWNLOAD LINK- click this to start your file download

Don't change the filename - the app needs these filenames to work

Remember where you saved this data!

6) Using the data in the app

You still need to upload to a fileserver to access the data in the app

Take the downloaded files and send them to whoever is hosting your webserver

The address of the files is as follows- they must be in a "docs" folder which is in the speciality folder...

eg. ent or plastics etc

The specialty folders are found at the "root" address which is used in the app.

All of the URL must be in lowercase (the app will only accept lowercase)
